# CASES GUIDE — PROJETOS / PORTFOLIO

## Objetivo da seção
Transformar os projetos em uma galeria de autoridade premium, com sensação editorial e curadoria estratégica.

A seção não deve parecer uma grade comum de portfólio.

Deve comunicar:
- autoridade
- profundidade estratégica
- domínio técnico
- percepção de marca premium
- capacidade de construir sistemas de embalagem

## Hierarquia dos cases

### CASE 01 — PURESOLO
Papel:
Case protagonista principal.

Descrição:
Sistema de embalagens desenvolvido para fortalecer percepção de valor, organizar portfólio e criar reconhecimento entre diferentes categorias de produtos.

Tratamento visual:
- card grande
- destaque máximo
- vídeo dominante
- sensação de case hero

### CASE 02 — SUPER HÚMUS
Papel:
Case protagonista premium.

Descrição:
Arquitetura visual estratégica para uma linha de embalagens pensada para gerar reconhecimento, coerência e valor percebido no ponto de venda.

Tratamento visual:
- card grande
- deslocamento editorial
- forte presença visual
- sensação orgânica e premium

### CASE 03 — GG RAÇÕES
Papel:
Case complementar.

Descrição:
Arquitetura visual para nutrição animal por fase de produção, com foco em clareza comercial, diferenciação por categoria e reconhecimento imediato.

Tratamento visual:
- card menor
- apoio estratégico
- demonstra diversidade de atuação
- não deve competir com Puresolo e Super Húmus

## Layout recomendado
- Staggered layout editorial
- Puresolo e Super Húmus com maior destaque
- GG Rações menor
- Cards com vídeos WEBM
- Hover cinematográfico

## Linguagem dos cases
Os textos devem ser objetivos, premium e estratégicos.

Evitar:
- linguagem promocional exagerada
- textos longos
- tom de anúncio
- aparência de catálogo
