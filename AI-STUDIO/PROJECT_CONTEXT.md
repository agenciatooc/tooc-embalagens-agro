# TOOC LANDING PAGE — PROJECT CONTEXT

## Projeto
Agência TOOC — Landing page premium para branding estratégico e design de embalagens para o agro.

## Posicionamento
70% agência criativa premium  
30% plataforma estratégica para o agro

## Objetivo principal
Posicionar a Agência TOOC como especialista high-end em design de embalagens, branding estratégico e arquitetura visual para empresas do agro.

## Referências visuais
- Obah Design
- Apple cinematic presentation
- Luxury editorial websites
- Premium motion design
- Branding studios internacionais

## Linguagem visual
- Interface escura e premium
- Atmosfera cinematográfica
- Pouco ruído visual
- Hierarquia forte
- Tipografia grande
- Espaçamento editorial
- Uso sofisticado de área negativa
- Embalagens como protagonistas

## Personalidade da marca

## Regras importantes
Não redesenhar o projeto inteiro sem solicitação.
Preservar a estrutura aprovada.
Evoluir seção por seção.
Manter linguagem premium, editorial e cinematográfica.
Evitar aparência genérica de agência comum.

## Stack
- HTML
- CSS
- JavaScript
- Vídeos WEBM
- Google Flow para geração de vídeos
- Google AI Studio / Gemini para apoio de front-end
- VS Code como ambiente de validação e produção

## Direção criativa
O site deve parecer uma landing page de estúdio premium de branding e packaging, com forte domínio do agro e foco em autoridade visual.
